# jQuery Practice

This repository contains my hands-on practice and implementation of core jQuery concepts. It showcases different functionalities like events, effects, animations, and DOM manipulation using jQuery.

---

## 📌 Topics Covered

### 1. Document Ready & Alert
- Understanding how to use `$(document).ready()`
- Displaying alerts using jQuery

### 2. Click Events
- Handling button click events
- Executing functions on user interaction

### 3. Keyboard Events
- Detecting key actions using `keyup`
- Working with real-time user input

### 4. Effects
- hide()
- show()
- toggle()
- slideUp()
- slidedown()
- Fadein()
- Fadeout()
- FadeToggle()

### 5. Animation & Stop
- animate()
- stop()
- Controlling animations using buttons

### 6. Show/Hide Box
- Dynamically showing and hiding elements
- Basic UI interaction

### 7. Method Chaining & Styling
- Applying multiple jQuery methods together
- Adding styles like borders using chaining

### 8. Add & Remove Class
- addClass()
- removeClass()
- Dynamically changing element styles
